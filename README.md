# Auth-Service
JWT를 이용한 인증 서비스 개인 프로젝트 

<br>

### 기술스택
🟧 <b>Backend</b>  
- Java 11  
- Spring Boot 2.7.6  
- MySQL  
- Spring Data JPA  
- Redis  
- Junit5  
- H2  

<br>

🟨 <b>Frontend</b>  
- HTML  
- CSS  
- JavaScript  
- JQuery  
- BootStrap  
- Thymeleaf
